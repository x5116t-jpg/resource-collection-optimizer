タイトル: 地域交通計画立案ツールを改良した群馬県内の
         未利用資源運搬等に関する基礎的研究

作成者: 独立行政法人自動車技術総合機構
        交通安全環境研究所

====================================================================
コンパイル方法
====================================================================

方法1: バッチファイルを使用（推奨）
-----------------------------------
1. compile.bat をダブルクリック
2. コンパイルが完了すると main.pdf が生成されます
3. PDFを開くか選択できます

方法2: コマンドラインから手動で実行
-----------------------------------
1. コマンドプロンプトでこのディレクトリに移動
2. 以下のコマンドを順番に実行:
   platex main.tex
   platex main.tex
   dvipdfmx main.dvi

====================================================================
ファイル構成
====================================================================

メインファイル:
  main.tex                          - メイン文書ファイル
  compile.bat                       - コンパイル用バッチファイル
  clean.bat                         - 中間ファイル削除用バッチファイル

セクションファイル:
  section_2_1_system_overview.tex   - システムの全体構成
  section_2_2_details.tex           - システムの詳細（親）
  section_2_2_1_resources.tex       - 資源の説明
  section_2_2_2_vehicles.tex        - 車両の説明
  section_2_2_3_compatibility.tex   - 適合性表
  section_2_2_4_optimization.tex    - 最適化アルゴリズム
  section_2_2_5_cost.tex            - コスト計算
  section_3_usage.tex               - 使い方
  section_4_conclusion.tex          - 結論

補助ファイル:
  references.tex                    - 参考文献
  appendix_a.tex                    - 付録A（システム設計の詳細）
  appendix_c.tex                    - 付録C（技術的補足）
  compatibility_table.tex           - 適合性マトリックス表

生成ファイル:
  main.pdf                          - 最終PDF文書
  main.aux, main.log, main.toc等   - LaTeX中間ファイル

====================================================================
トラブルシューティング
====================================================================

Q: コンパイルエラーが出る
A: 以下を確認してください:
   - TeX Live 2025がインストールされているか
   - すべての.texファイルがUTF-8エンコーディングになっているか
   - clean.batで中間ファイルを削除してから再コンパイル

Q: 表が紙面からはみ出す
A: compatibility_table.texの表は幅が広いため、
   必要に応じて横向きページに変更してください

Q: PDFが生成されない
A: エラーメッセージを確認し、.logファイルを確認してください
