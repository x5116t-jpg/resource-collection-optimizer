## 💡 よくある質問

### Q: どのドキュメントから読めばいいですか？

**A**: 用途によって異なります：

- **すぐに使いたい** → [クイックスタートガイド](quickstart_guide.md)
- **詳しく知りたい** → [使い方マニュアル](user_manual.md)
- **発表資料が必要** → [system_presentation.pptx](system_presentation.pptx)
- **仕組みを理解したい** → [system_flowchart_explanation.md](system_flowchart_explanation.md)

### Q: ドキュメントを編集できますか？

**A**: はい、すべて編集可能です：

- **Markdown**: テキストエディタで編集
- **PowerPoint**: PowerPointまたはGoogleスライドで編集
- **SVG**: Illustrator、Inkscape等で編集

### Q: PDFに変換できますか？

**A**: はい：

- **Markdown**: Typora、Pandoc等で変換
- **PowerPoint**: PowerPointから「PDFとして保存」
- **HTML**: ブラウザから印刷 → PDF保存

### Q: プレゼン資料をカスタマイズしたい

**A**: [presentation_usage_guide.md](presentation_usage_guide.md) を参照してください。
色、フォント、レイアウトの変更方法を説明しています。
