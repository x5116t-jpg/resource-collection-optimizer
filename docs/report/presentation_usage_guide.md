## 🎯 使用方法

### PowerPoint（system_presentation.pptx）の使い方

#### 基本的な使い方
1. **開く**: ファイルをダブルクリック、または PowerPoint から開く
2. **編集**: 各スライドのテキストやレイアウトを自由に編集
3. **プレゼン**: スライドショーモード（F5キー）で発表

#### Googleスライドで開く方法
1. Googleドライブにアクセス
2. 「新規」→「ファイルのアップロード」
3. `system_presentation.pptx` を選択してアップロード
4. アップロード完了後、ファイルを右クリック
5. 「アプリで開く」→「Googleスライド」を選択
6. 自動的にGoogleスライド形式に変換されます

#### カスタマイズのヒント
- **色の変更**: 図形を選択 → 「書式」タブ → 「図形の塗りつぶし」
- **フォント変更**: テキストを選択 → 「ホーム」タブ → フォント選択
- **スライド追加**: 「ホーム」タブ → 「新しいスライド」
- **画像挿入**: 「挿入」タブ → 「画像」

---

### HTML（system_presentation.html）の使い方

#### ブラウザで表示
1. ファイルをダブルクリック
2. デフォルトブラウザで開きます
3. スクロールして各スライドを確認

#### PDFに変換
1. ブラウザで開く（Chrome推奨）
2. `Ctrl + P`（印刷）
3. 「送信先」で「PDFに保存」を選択
4. 「保存」をクリック

#### Web公開
1. HTMLファイルをWebサーバーにアップロード
2. URLをシェア
3. どのデバイスからでも閲覧可能

---

### SVG形式（フロー図）の使い方

#### ブラウザで表示
- ファイルをダブルクリック → 自動的にブラウザで開きます

#### 編集する場合
1. **Adobe Illustrator**
   - ファイル → 開く → SVGファイルを選択
   - ベクターグラフィックとして編集可能

2. **Inkscape（無料）**
   - https://inkscape.org/ からダウンロード
   - ファイル → 開く → SVGファイルを選択
   - 色、テキスト、レイアウトを自由に編集

3. **PowerPointに挿入**
   - PowerPointを開く
   - 「挿入」タブ → 「画像」
   - SVGファイルを選択
   - 拡大縮小しても劣化しません

4. **Wordに挿入**
   - Wordを開く
   - 「挿入」タブ → 「画像」
   - SVGファイルを選択

## 🎨 カスタマイズ例

### 会社ロゴの追加（PowerPoint）
1. スライド1（タイトル）を開く
2. 「挿入」→「画像」でロゴを挿入
3. サイズと位置を調整
4. すべてのスライドに適用する場合:
   - 「表示」→「スライドマスター」
   - マスターに画像を配置
   - 「スライドマスター」タブ → 「マスター表示を閉じる」

### 色テーマの変更（PowerPoint）
1. 「デザイン」タブ → 「バリエーション」
2. 「色」→ カラーセットを選択
3. またはカスタムカラーを作成

### フォントの統一変更（PowerPoint）
1. 「ホーム」タブ → 「置換」→「フォントの置換」
2. 置換前と置換後のフォントを選択
3. 「置換」をクリック

## 🔧 トラブルシューティング

### PowerPointが開けない
- **原因**: Microsoft Office がインストールされていない
- **解決策**:
  1. Googleスライドで開く（上記参照）
  2. LibreOffice Impress（無料）を使用
  3. PowerPoint Viewerをインストール

### SVGが表示されない
- **原因**: 古いブラウザまたはソフトウェア
- **解決策**:
  1. 最新のブラウザ（Chrome、Firefox、Edge）を使用
  2. ブラウザで開いてスクリーンショット
  3. PNGに変換して使用

### HTMLの印刷がうまくいかない
- **解決策**:
  1. Google Chrome を使用
  2. 「印刷」設定で「背景のグラフィック」をON
  3. 余白を「最小」に設定
